## What is Sulaf?

Sulaf is a **shadcn-vue registry** designed to bring premium aesthetics and world-class developer experience to the Vue ecosystem. It's not a component library you install; it's a collection of high-quality components that you copy and paste directly into your project.

## Project Structure

- `apps/www`: The documentation site and registry showcase.
- `packages/registry`: The core component registry data and distribution.
- `packages/ui`: all shadcn-vue components.
